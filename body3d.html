The Anatomy

A 3D articulated figure whose hips, knees, and ankles move on the same literature-grounded joint-angle curves as every other page — and whose muscles glow from cool to hot with their real activation levels. Orbit around it, scrub the cycle, and watch which muscles do the work through the movement. Built in three.js (WebGL) running straight in your browser.

This build uses stylised procedural bone & muscle volumes as a stand-in. The rendering and data pipeline is designed so a real anatomical mesh (e.g. an open Z-Anatomy / BodyParts3D model) can be dropped in later without changing the data wiring.

The same six activation curves, painted onto a 2D anatomical map (front + back) as an EMG-style heatmap — neutral → yellow → orange → red with rising activation. Unlike the 3D figure it doesn't move with the joints; it's a body that lights up.

Both views read from the same activation curves at the current instant — the 3D muscle volumes glow cool → hot, the 2D map lights up neutral → red. Running activation timing is grounded in the Santuz et al. 2018 open running-EMG data set; see Sources for the rest. The 2D map is the open body-muscles library (Apache 2.0); the 3D figure is three.js.
